# Smart Helmet Safety System

## Overview

This project is an Arduino Uno based smart safety system for two-wheeler riders.

## Features

- Helmet Wear Detection
- Alcohol Detection
- Fall Detection using MPU6050
- Buzzer Alert
- LCD Display Monitoring

## Components Used

- Arduino Uno
- MPU6050
- Alcohol Sensor
- 16x2 LCD Display
- Limit Switches
- Buzzer
- LED

## Working

1. Rider wears helmet.
2. Helmet switches verify helmet usage.
3. Alcohol sensor checks rider condition.
4. Bike starts only if safety conditions are satisfied.
5. MPU6050 detects accidents.
6. Alert system activates during emergency.
